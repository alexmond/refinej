== Vision & Objectives

RefineJ is the semantic refactoring CLI that finally lets AI coding agents perform safe rename/move operations on Java projects.

**MVP Goal**: A production-ready, pluggable CLI that indexes a Java codebase and executes rename/move with preview/apply in under 5 minutes on a 5k-file repo.

**Total MVP Timeline**: 8–10 weeks (1–2 developers)
**Release Target**: v0.1.0 (open-source on GitHub, executable JAR + native binary)

== Prioritization Legend

* *P0 (Critical)* — Must be done for any usable MVP
* *P1 (High)* — Required for core value
* *P2 (Medium)* — Nice-to-have for v0.1
* *P3 (Future)* — Post-MVP (v0.2+)

== How to Turn This Roadmap into Tickets

3. Add labels: `phase-0`, `phase-1`, … `priority-P0`, etc.
4. Add to a GitHub Project board (columns: Backlog → In Progress → Review → Done).
